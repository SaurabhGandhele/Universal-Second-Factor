Files you need to add to the prefido.war to make it post-fido:

JSP files to be added to 'FIDOTutorial/prefido/web/'
1. fidoauthenticate.jsp
2. fidoregister.jsp

Add JS files to prefido/web/js:
1. common.js
2. enrolldata.js
3. FIDOAJAXMethods.js
4. jquery-2.0.0.min.js
5. u2f-api.js

Add images to prefido/web/img:
1. loading.gif

Java files to be added to 'FIDOTutorial/prefido/src/java/com/strongauth/fidotutorial/prefido'
1. FIDOAJAXServlet.java
2. SKFERestClient.java

In addition to this, you will need the following JARs which can be downloaded from the Internet
1. httpclient-4.5.1.jar
2. httpcore-4.4.jar
